# Taller 4 — APIs Públicas, MongoDB y EDA

## Descripción
Pipeline de Ciencia de Datos que consume la Dragon Ball API, almacena los datos crudos en MongoDB y realiza un análisis exploratorio (EDA) en Jupyter Notebook.

## API utilizada
**Dragon Ball API** — https://web.dragonball-api.com/

Datos de personajes, planetas y transformaciones del universo Dragon Ball.

Total registros: 121 documentos.

## Insights encontrados
1. Los **Saiyans** son la raza más frecuente con 10 personajes.
2. El universo Dragon Ball tiene **13 razas distintas**.
3. El **87.9%** de los personajes son masculinos, solo 7 son femeninos.
4. La afiliación más común es **Z Fighter** con 19 personajes.
5. Hay **8 afiliaciones distintas**, desde héroes hasta villanos.
